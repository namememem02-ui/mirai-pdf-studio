# PDF Support 📄

เว็บเครื่องมือจัดการ PDF — **ประมวลผลในเบราว์เซอร์ 100% ไฟล์ไม่ถูกอัปโหลดขึ้นเซิร์ฟเวอร์**

## เครื่องมือ

| เมนู | ทำอะไร |
|---|---|
| 🗂️ รวม PDF | รวมหลายไฟล์เป็นไฟล์เดียว จัดลำดับได้ |
| ✂️ แยกหน้า | เลือกหน้า เช่น `1-3,5` ออกเป็นไฟล์ใหม่ |
| 🔄 หมุนหน้า | หมุน 90/180/270° ทุกหน้าหรือเฉพาะบางหน้า |
| 🖼️ รูปภาพ → PDF | JPG/PNG หลายรูป → PDF ไฟล์เดียว |
| 📸 PDF → รูปภาพ | ทุกหน้าเป็น PNG โหลดทีละหน้า/ZIP |
| 📝 ดึงข้อความ | คัดลอกหรือบันทึกเป็น .txt |

## วิธีเปิดใช้

ดับเบิลคลิก **start.bat** (เปิดที่ http://localhost:4200)

หรือ: `npm install` ครั้งแรก แล้ว `npm run dev`

## วิธีเพิ่มเครื่องมือใหม่

1. เพิ่มรายการใน `src/lib/tools.ts`
2. สร้าง `src/app/<id>/page.tsx` (copy โครงจากเครื่องมือที่ใกล้เคียง)

ทุกเมนูแยกไฟล์กันเด็ดขาด แก้ตัวหนึ่งไม่กระทบตัวอื่น

## เทคโนโลยี

Next.js 16 + pdf-lib (สร้าง/แก้ PDF) + pdfjs-dist (อ่าน/เรนเดอร์) + jszip — ไม่มี API route ไม่มี server processing จึง deploy เป็นเว็บ static บน Vercel ฟรีได้ทันที
